Get single player

If the variable in the url is not present it will be redirected to this endpoint.

If session header are not present or incorrect this endpoint will return only partial content (206).

/players/{$username}

Get player full data. Return only simple data if session is incorrect or isn't entered.

Responses

{ "id": 1, "username": "username1" }
{ "message": "Player doesn't exists", "file": "players.php", "line": 0 }

HTTP Errors - 404
