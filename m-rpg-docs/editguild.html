Edit guild

There is a variable present in the url of this endpoint. If the variable is not present while using this endpoint, it will return this error.

This endpoints requires user to be logged in. If any of the session headers is not present it will return error 400. If session is incorrect it will return error 401. In order to acquire session key go here.

There must be at least one property in the request body, otherwise it will throw a 400 error.

/guilds/{$slug}

Select new leader.

Request parameters

{ "leader": "username1", "vice_leader": "username2" }

Responses

{ "message": "Incorrect request", "file": "guilds.php", "line": 0 }
{ "message": "Unauthorized", "file": "guilds.php", "line": 0 }
{ "message": "Not found", "file": "guilds.php", "line": 0 }

HTTP Errors - 400

Enter guilds name

Occurs when there is no guild name in the URL.

Enter some changes

Occurs when there are no changes in the request body.

HTTP Errors - 401

You don't have permission to do this (vice leader)

You have to be at least guilds vice leader to do this.

You don't have permission to do this (leader)

You have to be at least guilds leader to do this.

You can't downgrade your leader

You can not downgrade your guilds leader to vice leader. Executed from vice leaders POV.

HTTP Errors - 404

Guild doesn't exists

The entered guild does not exist.

Player doesn't exists (leader)

The entered player does not exist. Threw by new leader.

Player doesn't exists (vice leader)

The entered player does not exist. Threw by new vice leader.
