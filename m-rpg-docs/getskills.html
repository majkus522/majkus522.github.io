Get player skills

There is a variable present in the url of this endpoint. If the variable is not present while using this endpoint, it will return this error.

This endpoints requires user to be logged in. If any of the session headers is not present it will return error 400. If session is incorrect it will return error 401. In order to acquire session key go here.

/skills/{$username}

Get player skills.

Request parameters

Responses

[ { "id": 1, "player": 1, "skill": "skill1", "toggle": 0 }, { "id": 1, "player": 1, "skill": "skill2", "toggle": 1 } ]
{ "message": "Incorrect request", "file": "skills.php", "line": 0 }
{ "message": "Not found", "file": "skills.php", "line": 0 }

HTTP Errors - 400

Enter player

Occurs when there is no player username in the URL.

Unknown query string parameter {$parameter}

Occurs when the {$parameter} is invalid.

Unknown order parameter {$value}

Occurs when the order query string has incorrect value.

Incorrect query string (toggle) value

Occurs when the toggle query string has incorrect value.

Unknown rarity parameter {$value}

Occurs when the rarity query string has incorrect value.

HTTP Errors - 404

Player doesn't exists

The entered player does not exist.

Can't find any skills matching conditions

Can't find any skill matching query string parameters to given player.
