Toggle players skill

There is a variable present in the url of this endpoint. If the variable is not present while using this endpoint, it will return error 400.

This endpoints requires user to be logged in. If any of the session headers is not present it will return error 400. If session is incorrect it will return error 401. In order to acquire session key go here.

/skills/{$username}/{$skill}

Toggle players skill.

Request parameters

true
false

Responses

{ "message": "Error message" }
{ "message": "Error message" }

Possible errors - 400

  • Skill can't be toggled

  • Enter new toggle value

Possible errors - 404

  • Player doesn't exists

  • Skill doesn't exists
