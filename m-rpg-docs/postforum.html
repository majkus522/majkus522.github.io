This endpoints requires user to be logged in. If any of the session headers is not present it will return error 400. If session is incorrect it will return error 401. In order to acquire session key go here.

/forum

Create a new post or comment an existing post.

Request parameters

{ "text": "Lorem ipsum", "title": "Title" }
{ "text": "Lorem ipsum", "master": "1" // ID of master comment }

Responses

{ "id": "1", // Appears only if new comment is created. "slug": "slug-1" // Appears only if new post is created. }
{ "message": "Incorrect request", "file": "forum.php", "line": 0 }

HTTP Errors - 400

Enter text content

There is no text content in request body.

Comment or post doesn't exists

Master comment of new comment does not exist.

Title is too short

New post title should be at least 3 characters long.

Enter post title or parent comment

There is no title or master comment in request body.
