Check player session

There is a variable present in the url of this endpoint. If the variable is not present while using this endpoint, it will return error 400.

/players/{$username}/session

Check if player session is correct.

Request parameters

Responses

{ "message": "Incorrect request", "file": "players.php", "line": 0 }
{ "message": "Unauthorized", "file": "players.php", "line": 0 }
{ "message": "Player doesn't exists", "file": "players.php", "line": 0 }

Possible errors - 400

  • Enter session key

  • Enter session id

Possible errors - 401

  • Incorrect player

  • Incorrect session key
