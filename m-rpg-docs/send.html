Send recovery email

Only one of the Request Body properties is required.

/password-recovery

Sends password recovery email to player.

Request parameters

{ "username": "username1", "email": "example@mail.com" }

Responses

{ "message": "Enter username or email", "file": "password-recovery.php", "line": 0 }
{ "message": "Player doesn't exists", "file": "password-recovery.php", "line": 0 }

HTTP Errors - 400

Enter username or email

Occurs when the email address or username property is missing in the request body.

HTTP Errors - 404

Player doesn't exists

The entered player does not exist.
