Login player

There is a variable present in the url of this endpoint. If the variable is not present while using this endpoint, it will return this error.

/players/{$username}/login

Login player and get session key.

Request parameters

Responses

{ "key": "0874766202401092016080742134", "id": 0 }
{ "message": "Incorrect request", "file": "players.php", "line": 0 }
{ "message": "Wrong password", "file": "players.php", "line": 0 }
{ "message": "Player doesn't exists", "file": "players.php", "line": 0 }

HTTP Errors - 400

Enter player

Occurs when there is no player username in the URL.

Enter player password

Occurs when there is no Password header in the request.

Enter session type

Occurs when there is no Session-Type header in the request.

HTTP Errors - 401

Wrong password

Player password is incorrect.

HTTP Errors - 404

Player doesn't exists

The entered player does not exist.
