Create new player

/players

Register new player. Automatically logins player and returns session key.

Request parameters

{ "username": "username1", "password": "UEBzc3cwcmQ=", //Must be encoded using BASE64 "email": "example@mail.com" }

Responses

{ "key": "0874766202401092016080742134", "id": 0 }
{ "message": "Error message", "file": "players.php", "line": 0 }
