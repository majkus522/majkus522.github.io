Create new player

/players

Register new player. Automatically logins player and returns session key.

Request parameters

{ "username": "username1", "password": "UEBzc3cwcmQ=", //Must be encoded using BASE64 "email": "example@mail.com" }

Responses

1234567890123456789012345678
{ "message": "Error message", "file": "players.php", "line": 0 }

Possible errors - 400

Username errors

  • Enter username

  • Player already exists

  • Username is too short

  • Username is too long

  • Username can only contain letters, numbers and underscore

Email errors

  • Enter email

  • Email already taken

  • Incorrect email

Password errors

  • Enter password

  • Password must be at least 6 characters long

  • Password must contain at least one large character

  • Password must contain at least one small character

  • Password must contain at least one number

  • Password must contain at least one special character
