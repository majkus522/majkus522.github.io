Create guild

This endpoints requires user to be logged in. If any of the session headers is not present it will return error 400. If session is incorrect it will return error 401. In order to acquire session key go here.

/guilds

Create own guild.

Request parameters

{ "name": "Name 3", "leader": "username3" }

Responses

{ "message": "Incorrect request", "file": "guilds.php", "line": 0 }

HTTP Errors - 400

Enter guilds name

There is no guild name in request body.

Enter guilds leader

There is no guild leader in request body.

Guilds name is too long

Guilds name should contain max 100 characters.

Guild already exists

Guilds name is already taken.

You are already part of guild

You can not be part of any guild to create own.
