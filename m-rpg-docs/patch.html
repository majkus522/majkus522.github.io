Set players new password

There is a variable present in the url of this endpoint. If the variable is not present while using this endpoint, it will return error 400.

/password-recovery/{$code}

Sets a new password for users who have forgotten it.

Request parameters

{ "password": "bmV3IHBhc3N3b3Jk" //Must be encoded using BASE64 }

Responses

{ "message": "Error message", "file": "password-recovery.php", "line": 0 }
{ "message": "Link expired", "file": "password-recovery.php", "line": 0 }

Possible errors - 400

  • Enter code

Password errors

  • Enter password

  • Password must be at least 6 characters long

  • Password must contain at least one large character

  • Password must contain at least one small character

  • Password must contain at least one number

  • Password must contain at least one special character
