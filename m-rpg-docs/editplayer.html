Edit player data

There is a variable present in the url of this endpoint. If the variable is not present while using this endpoint, it will return this error.

This endpoints requires user to be logged in. If any of the session headers is not present it will return error 400. If session is incorrect it will return error 401. In order to acquire session key go here.

/players/{$username}

Change player data. At least one property must be present in the request body, otherwise it will throw a error 400.

Request parameters

{ "password": "new password", //Must be encoded using BASE64 "email": "new email" }

Responses

{ "message": "Error message", "file": "players.php", "line": 0 }

HTTP Errors - 400

Enter player

Occurs when there is no player username in the URL.

Enter some changes

Occurs when there are no changes in the request body.

Enter email

The email must be included in the request body.

Email already taken

Email is already taken.

Incorrect email

Entered email is invalid.

Enter password

The password must be included in the request body.

Password must be at least 6 characters long

The new password must contain at least 6 characters.

Password must contain at least one large character

The new password must contain at least 1 uppercase character.

Password must contain at least one small character

The new password must contain at least 1 lowercase character.

Password must contain at least one number

The new password must contain at least 1 digit.

Password must contain at least one special character

The new password must contain at least 1 special character.
