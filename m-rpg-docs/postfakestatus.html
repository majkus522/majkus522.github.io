Create fake status

There is a variable present in the url of this endpoint. If the variable is not present while using this endpoint, it will return this error.

This endpoints requires user to be logged in. If any of the session headers is not present it will return error 400. If session is incorrect it will return error 401. In order to acquire session key go here.

/fake-status/${username}

Create fake status for player.

Request parameters

{ "level": 1, "money": 1, "clazz": "archer", "str": 1, "agl": 1, "chr": 1, "intl": 1, "def": 1, "vtl": 1, "dex": 1 }

Responses

{ "message": "Incorrect request", "file": "fake-status.php", "line": 0 }
{ "message": "Not found", "file": "fake-status.php", "line": 0 }

HTTP Errors - 400

Enter player

Occurs when there is no player username in the URL.

Enter $parameter

Occurs when the {$parameter} is missing in the request body

Incorrect value: $parameter

Occurs when the {$parameter} value is invalid.

HTTP Errors - 404

Player doesn't exists

The entered player does not exist.

Player doesn't have fake status

The introduced player does not have the fake status skill.
